# TODO: Реализация двухрежимного бота (ЛС + беседы)

## Цель
Создать VK-бота, работающего одновременно в двух режимах:
- **Личные сообщения (ЛС)** — интерфейс для пользователей (покупка и управление рекламой)
- **Беседы** — текущий функционал для админов и разработчика

### 3. Реализация ЛС-интерфейса
#### Первое сообщение в ЛС

- [ ] Кнопки:
  - 📢 Моё сообщение
  - 💰 Купить рассылку
  - ✏️ Редактировать текст
  - 🖼️ Редактировать фото
  - ℹ️ Помощь

#### Команда `.сообщение` (в ЛС)
- [ ] Показать текущее активное сообщение пользователя
- [ ] Формат:
```
📨 ВАШЕ СООБЩЕНИЕ В РАССЫЛКЕ

{текст пользователя}

🖼️ Фото: прикреплено / не прикреплено
🕒 Активно до: 2026-05-19 18:00
🔄 Отправляется в: 147 чатах
💵 Стоимость: 450 ₽ (месяц)
```
- [ ] Если нет активного заказа — предложить нажать «Купить рассылку»

#### Команда `.купить` (в ЛС)
- [ ] Отправить клавиатуру с выбором тарифа (`generate_rates_menu()`)
- [ ] При выборе тарифа:
  - Сгенерировать `order_id` (например, `order_123`)
  - Сохранить заявку в `orders.json` со статусом `pending`
  - Отправить сообщение:
    ```
    💳 Оплатите {сумма} ₽ на Сбер: 2202 22**
    📎 Пришлите чек (фото или файл)
    ```

#### Прием вложений (чека)
- [ ] При получении фото или файла от пользователя:
  - Сохранить `attachment` (photo, doc)
  - Обновить заявку в `orders.json`
  - Отправить в админ-чат уведомление:
    ```
    📥 НОВАЯ ЗАЯВКА НА РАССЫЛКУ
    Используйте команду .заказы для просмотра
    ```

### 4. Админ-панель (.заказы)

#### Кнопки модерации
- [ ] ✅ **Одобрить**:
  - Обновить статус → `approved`
  - Сохранить сообщение в `ads.json` как `active_ad`
  - Отправить пользователю:
    ```

- [ ] ❌ **Отклонить**:

    ```

  - Переслать вложение из заявки как сообщение

- [ ] 🔚 **Выйти**:
  - Редактировать сообщение обратно в список заявок

### 5. Команды редактирования (.редсообщение, .редфото)
#### `.редсообщение <текст>` (в ЛС)
- [ ] Приостановить текущую рассылку данного сообщения

  ```json

  ```
- [ ] Отправить уведомление в админ-чат (аналогично новой заявке)

#### `.редфото` (в ЛС)
- [ ] Ответить: `📎 Пришлите новое фото`
- [ ] После получения:
  - Сохранить вложение
  - Создать заявку `update_ad`
  - При одобрении — обновить и возобновить рассылку
